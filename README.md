# Word-Counter
A terminal based Java word counter that inly works on .txt files.
Upon running the code you will be requested to input the file path to the file that you want to count.
After inputing the file path the code will read each line. 
Once it finished processing the file, ypu will be requested an output format -- alphabetically or by frequency (most to least).
It will output the word count, list the frequency of each word, and the number of lines.
