# Word-Counter
A terminal-based Java word counter that only works on .txt files.
Upon running the code, you will be required to input the file path to the file you want to count.
After inputting the file path, the code will read each line. 
Once it finishes processing the file, you will be requested to choose an output format—alphabetically or by frequency (most to least).
It will output the word count, list the frequency of each word, and give the number of lines.
